Como funciona
Três etapas, zero instalação
01

Selecione o PDF unificado

Arraste ou clique para escolher o arquivo com todos os comprovantes agrupados em um único PDF. O processamento é 100% local — nenhum dado sai do seu navegador.

02

Identificação automática por página

Cada página é lida via PDF.js e o tipo de comprovante é detectado por padrões de texto (DARE, pagamento normal, ICMS/parcelamento). O nome do arquivo é montado com os dados extraídos.

03

Download do ZIP com tudo separado

Ao finalizar, um arquivo .zip é gerado com todos os PDFs individuais já nomeados corretamente. Páginas não identificadas são incluídas como pagina_XX_nao_identificada.pdf.

Mapeamentos suportados

DARE — SEFAZ-SP

Detecta o cabeçalho SEFAZ-SP/DARE, extrai o ID do comprovante e o valor.

→ DARE - [ID] - [valor].pdf
DARE

Comprovante de pagamento normal

Busca "Valor pago" e o campo "Pagador", ignorando ocorrências precedidas por CPF/CNPJ.

→ [Pagador] - [valor].pdf
NORMAL

Comprovante ICMS / Parcelamento atualizado

Detecta PARCELAMENTO e VALOR DA RECEITA. Busca prefixos de aeroporto (BSAO, BCPQ, BCWB, BITJ, BSSA, BRRIO, BPOA) e captura o código completo de 11 caracteres. Se não encontrar, usa os últimos 10 dígitos do número de parcelamento.

→ BSAO0049698 - [valor].pdf  |  DI 0000000000 - [valor].pdf
